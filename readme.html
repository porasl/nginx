Compiling NGINX with RTMP manually

Install dependencies

sudo apt update

sudo apt install -y build-essential libpcre3 libpcre3-dev libssl-dev zlib1g-dev

Download NGINX source code

cd /usr/local/src

sudo wget http://nginx.org/download/nginx-1.24.0.tar.gz

sudo tar -zxvf nginx-1.24.0.tar.gz

cd nginx-1.24.0

Clone the RTMP module

sudo git clone https://github.com/arut/nginx-rtmp-module.git

Configure NGINX build with RTMP

sudo ./configure --with-http_ssl_module --add-module=../nginx-rtmp-module

 Build and install

sudo make

sudo make install

This installs nginx to /usr/local/nginx/ by default.

Now Configure NGINX to enable RTMP

sudo nano /etc/nginx/nginx.conf

Or

sudo nano /usr/local/nginx/conf/nginx.conf

(depending on how you installed)

Add this inside http {} OR separately after events {}:

rtmp {

    server {

        listen 1935;  # RTMP default port

        chunk_size 4096;

        application live {

            live on;

            record off;

        }

    }

}

This enables an RTMP live streaming server on port 1935.

🚀 Start / Restart NGINX

sudo systemctl restart nginx

sudo /usr/local/nginx/sbin/nginx

To stream to your NGINX server:

Use OBS, or any RTMP-compatible encoder.

In OBS (or ffmpeg):

Server: rtmp://your-server-ip/live
Stream Key: whatever you want

Stream to:

rtmp://your-server-ip/live/mystream

then your stream will be available at:

rtmp://your-server-ip/live/mystream

Quick Troubleshooting

If NGINX fails to start: check syntax with nginx -t
Make sure port 1935 is open in your firewall (or cloud security group).
Use OBS to push stream → check logs at /var/log/nginx/error.log

how to set up NGINX RTMP to serve Public HLS streams

rtmp {

    server {

        listen 1935;

        chunk_size 4096;

        application live {

            live on;

            record off;

            hls on;

            hls_path /tmp/hls;

            hls_fragment 6s;

            hls_playlist_length 30s;

        }

    }

}

http {

    server {

        listen 8080;

        location /hls {

            types {

                application/vnd.apple.mpegurl m3u8;

                video/mp2t ts;

            }

            root /tmp;

            add_header Cache-Control no-cache;

            add_header Access-Control-Allow-Origin *;

        }

    }

}
